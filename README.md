# 🎯 Career Recommendation System

## 📌 Problem
Students often face confusion while choosing careers.

## 💡 Solution
This project uses Machine Learning to recommend careers based on skills and marks.

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Streamlit

## 🚀 How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Train model:
   python train.py

3. Run app:
   streamlit run app.py

## 📊 Features
- Predicts career using ML
- Simple UI
- Fast results

## 🔮 Future Improvements
- Add more careers
- Add personality test
- Improve accuracy
